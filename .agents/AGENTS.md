# Workspace Rules

- **Code wijzigingen locatie**: Voer wijzigingen ALTIJD alleen uit in de hoofdcode / productiemappen (bijv. root `index.html`, `js/`, `css/`). 
- **NIET aanpassen**: Pas NOOIT bestanden aan in de `beheer/` map of de `siteground_upload/` map. De gebruiker synchroniseert deze mappen zelf handmatig wanneer hij tevreden is met de wijzigingen in de hoofdcode.
